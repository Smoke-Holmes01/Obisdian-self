# 一、类和对象

### 1. 类的定义

​	具有相同特征和行为的共性的抽象

### 2. 类的组成

* 属性（成员变量）
  * 默认值：整数--0
    		小数--0.0
    		字符--\u0000
    		布尔--false
    		引用数据类型--null
* 行为（成员方法）
  * 定义：
    	拥有功能性代码的代码块

### 3. 对象的定义

​	指一类事物的具体实现，分析事物的特征和行为

### 4.对象的使用

​	Java官方定义的类，我们在创建对象时需要导包（import 包名.类名）【java.lang包除外】，然后通过new 关键字来创建对象，然后通过引用变量名来调用成员如果是我们自己写的类，且这两个类在同一个包下，则不需要导包
