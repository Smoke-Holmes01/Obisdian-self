### Arrays工具类中的*方法*

##### 1. toString遍历

遍历数组

**Arrays.toString(数组名称)**

```java
 public void testArrayMethod() {
        int[] arr = {1, 2, 3, 4, 5, 6, 7};
        // 遍历3
        System.out.println(Arrays.toString(arr));
    }
```

```powershell
[1,2,3,4,5,6,7]
```
