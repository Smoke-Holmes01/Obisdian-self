# 常用的等价无穷小

## 基本概念

若 $\displaystyle\lim_{x \to x_0} \frac{f(x)}{g(x)} = 1$，则称 $f(x)$ 与 $g(x)$ 在 $x \to x_0$ 时**等价无穷小**，记作：

$$f(x) \sim g(x) \quad (x \to x_0)$$

> **核心用途：求极限时进行替换，简化计算。**

⚠️ **注意**：等价无穷小替换**一般只在乘除法中使用**，加减法中不能随意替换（除非满足特定条件）。

---

## 一、$x \to 0$ 时的等价无穷小（最重要）

### 基本等价关系

| 序号 | 等价关系 | 备注 |
|:---:|:---|:---|
| 1 | $\sin x \sim x$ | |
| 2 | $\tan x \sim x$ | |
| 3 | $\arcsin x \sim x$ | |
| 4 | $\arctan x \sim x$ | |
| 5 | $\ln(1+x) \sim x$ | |
| 6 | $e^x - 1 \sim x$ | |
| 7 | $a^x - 1 \sim x \ln a$ | $a>0$ |
| 8 | $1 - \cos x \sim \dfrac{1}{2}x^2$ | |
| 9 | ==$(1+x)^\alpha - 1 \sim \alpha x$== | $\alpha \in \mathbb{R}$ |
| 10 | $x - \sin x \sim \dfrac{1}{6}x^3$ | 泰勒展开 |
| 11 | $\tan x - x \sim \dfrac{1}{3}x^3$ | |
| 12 | $x - \ln(1+x) \sim \dfrac{1}{2}x^2$ | |
| 13 | $e^x -1 - x \sim \dfrac{1}{2}x^2$ | |
| 14 | $\arctan x - x \sim -\dfrac{1}{3}x^3$ | |

### 特别常用组合

- $\sin x \sim \tan x \sim \arcsin x \sim \arctan x \sim x$
- $\ln(1+x) \sim e^x - 1 \sim x$
- $1-\cos x \sim \dfrac{x^2}{2}$

### 根式展开（来自公式 9）

| 展开 | 等价形式 |
|:---|:---|
| $\sqrt{1+x} - 1 \sim \dfrac{1}{2}x$ | $\alpha = \frac{1}{2}$ |
| $\sqrt[3]{1+x} - 1 \sim \dfrac{1}{3}x$ | $\alpha = \frac{1}{3}$ |
| $\dfrac{1}{\sqrt{1+x}} - 1 \sim -\dfrac{1}{2}x$ | $\alpha = -\frac{1}{2}$ |
| $\sqrt[n]{1+x} - 1 \sim \dfrac{1}{n}x$ | $\alpha = \frac{1}{n}$ |

---

## 二、推广形式

令 $\square$ 代表取极限时趋于 $0$ 的表达式（可以是 $x$，也可以是 $f(x)$）：

- $\sin \square \sim \square$
- $\tan \square \sim \square$
- $\ln(1+\square) \sim \square$
- $e^\square - 1 \sim \square$
- $1 - \cos \square \sim \dfrac{1}{2}\square^2$
- $(1+\square)^\alpha - 1 \sim \alpha \square$

---

## 三、$x \to 0^+$ 的无穷大比较

当 $x \to 0^+$ 时（用于判断阶数）：

$$e^{\frac{1}{x}} \gg x^{-\alpha} \gg \ln^n x \quad (\alpha > 0)$$

---

## 四、$x \to \infty$ 时的等价无穷大

| 关系 | 说明 |
|:---|:---|
| $\ln x \ll x^\alpha \ll a^x \ll x! \ll x^x$ | $x \to +\infty,\ \alpha>0,\ a>1$ |

---

## 五、使用技巧

### 1. 乘除可直接替换
$$\lim_{x\to 0} \frac{\sin x \cdot \tan x}{x^2} = \lim_{x\to 0} \frac{x \cdot x}{x^2} = 1$$

### 2. 加减法谨慎替换

❌ 错误：$\lim\limits_{x\to 0} \dfrac{\sin x - \tan x}{x^3}$ 中不能直接替换 $\sin x \sim x$、$\tan x \sim x$（分子变成 0，丢失精度）

✅ 正确：用泰勒展开保留高阶项
$$\sin x - \tan x \sim (x - \frac{x^3}{6}) - (x + \frac{x^3}{3}) = -\frac{x^3}{2}$$

### 3. 配合洛必达法则
先用等价无穷小化简，再对未定式用洛必达法则，事半功倍。
