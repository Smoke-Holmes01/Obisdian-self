# 常用公式

## 一、曲率与曲率半径

### 1. 曲率 (Curvature)

**定义：** 曲率是描述曲线弯曲程度的量，记为 $K$。

#### 1.1 显式函数 $y = f(x)$

曲线 $y = f(x)$ 的曲率公式为：

$$K = \frac{|y''|}{(1 + (y')^2)^{\frac{3}{2}}}$$

其中 $y' = \frac{dy}{dx}$，$y'' = \frac{d^2y}{dx^2}$。

#### 1.2 参数方程形式 $x = x(t),\; y = y(t)$

若曲线由参数方程 $\begin{cases} x = x(t) \\ y = y(t) \end{cases}$ 给出，则：

$$K = \frac{|x'(t) y''(t) - x''(t) y'(t)|}{\big( (x'(t))^2 + (y'(t))^2 \big)^{\frac{3}{2}}}$$

#### 1.3 极坐标形式 $r = r(\theta)$

$$K = \frac{|r^2 + 2(r')^2 - r r''|}{(r^2 + (r')^2)^{\frac{3}{2}}}$$

---

### 2. 曲率半径 (Radius of Curvature)

曲率半径是曲率的倒数，记为 $R$。

$$R = \frac{1}{K}$$

#### 2.1 显式函数 $y = f(x)$

$$R = \frac{(1 + (y')^2)^{\frac{3}{2}}}{|y''|}$$

#### 2.2 参数方程形式

$$R = \frac{\big( (x'(t))^2 + (y'(t))^2 \big)^{\frac{3}{2}}}{|x'(t) y''(t) - x''(t) y'(t)|}$$

---

### 3. 曲率圆 (Osculating Circle)

曲率圆是与曲线在某点处具有相同曲率和相同切线的圆。

- 圆心位于曲线在该点的法线上
- 半径为 $R = \frac{1}{K}$
- 圆心坐标 $(x_0, y_0)$ 满足：
  - 在 $y = f(x)$ 上点 $(x, y)$ 处：
    $$x_0 = x - \frac{y'(1 + (y')^2)}{y''}$$
    $$y_0 = y + \frac{1 + (y')^2}{y''}$$

# 常用定理

## 一、拉格朗日中值定理 (Lagrange Mean Value Theorem)

**定理描述：**
如果函数 $f(x)$ 满足：
1. 在闭区间 $[a, b]$ 上连续
2. 在开区间 $(a, b)$ 内可导

则在 $(a, b)$ 内至少存在一点 $\xi$，使得：

$$f'(\xi) = \frac{f(b) - f(a)}{b - a}$$

**几何意义：**
在连续且光滑的曲线上，至少存在一点，使该点的切线平行于两端点的连线。

**推论：**
若 $f'(x) \equiv 0$，则 $f(x) \equiv C$（常数）。

---

## 二、介值定理 (Intermediate Value Theorem)

**定理描述：**
如果函数 $f(x)$ 在闭区间 $[a, b]$ 上连续，且 $f(a) \neq f(b)$，则对于介于 $f(a)$ 与 $f(b)$ 之间的任意实数 $C$，在 $(a, b)$ 内至少存在一点 $\xi$，使得：

$$f(\xi) = C$$

**特殊情况（零点定理/根的存在定理）：**
如果函数 $f(x)$ 在闭区间 $[a, b]$ 上连续，且 $f(a) \cdot f(b) < 0$（即两端点异号），则在 $(a, b)$ 内至少存在一点 $\xi$，使得：

$$f(\xi) = 0$$

**几何意义：**
连续函数的图像在区间内不会跳过任何中间值，必能取到两端点之间的所有值。

---

## 三、罗尔定理 (Rolle's Theorem)

**定理描述：**
如果函数 $f(x)$ 满足：
1. 在闭区间 $[a, b]$ 上连续
2. 在开区间 $(a, b)$ 内可导
3. $f(a) = f(b)$

则在 $(a, b)$ 内至少存在一点 $\xi$，使得：

$$f'(\xi) = 0$$

**几何意义：**
在连续且光滑的曲线上，若两端点纵坐标相等，则至少存在一条水平切线。

**说明：**
罗尔定理是拉格朗日中值定理 $f(a) = f(b)$ 时的特殊情况。

---

## 四、柯西中值定理 (Cauchy Mean Value Theorem)

**定理描述：**
如果函数 $f(x)$ 和 $g(x)$ 满足：
1. 在闭区间 $[a, b]$ 上连续
2. 在开区间 $(a, b)$ 内可导
3. $g'(x) \neq 0$（在 $(a, b)$ 内）

则在 $(a, b)$ 内至少存在一点 $\xi$，使得：

$$\frac{f(b) - f(a)}{g(b) - g(a)} = \frac{f'(\xi)}{g'(\xi)}$$

**几何意义：**
参数方程形式下，存在一点使得切线方向与弦方向平行。

**说明：**
当 $g(x) = x$ 时，柯西中值定理退化为拉格朗日中值定理。
