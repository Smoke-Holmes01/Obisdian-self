### 3.1 SQL概述

#### 3.1.2 SQL的特点

SQL是一个综合性、功能极强同时又简洁易学的语言

SQL集数据定义、数据查询、数据操纵，数据控制等功能于一体

| SQL功能  |         动词         |
| :------: | :------------------: |
| 数据定义 | CREATE，DROP，ALTER  |
| 数据查询 |        SELECT        |
| 数据操纵 | INSERT,UPDATE,DEKETE |
| 数据控制 |     GRANT,REVOKE     |

#### 3.1.3 SQL的基本概念

SQL同样支持数据库系统的三级模式结构，其中

1. **外模式**是用户能够看见和使用的数据结构，对应==视图和部分基本表==，
2. **模式**对应==基本表==
3. **内模式**对应==存储文件==



### 3.2 数据定义



#### 3.2.2 基本表的删除定义和修改(mysql)

##### 1. 添加新列

```sql
ALTER TABLE 表名
ADD COLUMN 列名 数据类型 [约束] [AFTER 列名];
```

**示例**：添加年龄列

```sql
ALTER TABLE student1
ADD COLUMN Sage INT AFTER Ssex;
```

##### 2. 修改列定义

```sql
ALTER TABLE 表名
MODIFY COLUMN 列名 新数据类型 [新约束];
```

**示例**：将专业列长度改为 30

```sql
ALTER TABLE student1
MODIFY COLUMN Smajor VARCHAR(30);
```

##### 3. 重命名列

```sql
ALTER TABLE 表名
CHANGE COLUMN 旧列名 新列名 数据类型;
```

**示例**：重命名性别列

```sql
ALTER TABLE student1
CHANGE COLUMN Ssex Gender CHAR(1);
```

##### 4. 删除列

```sql
ALTER TABLE 表名
DROP COLUMN 列名;
```

**示例**：删除专业列

```sql
ALTER TABLE student1
DROP COLUMN Smajor;
```

##### 5. 添加约束

```sql
ALTER TABLE 表名
ADD CONSTRAINT 约束名 约束类型 (列名);
```

**示例**：添加检查约束（MySQL 8.0.16+）

```sql
ALTER TABLE student1
ADD CONSTRAINT chk_gender CHECK (Gender IN ('男', '女'));
```

##### 6. 删除约束

```sql
ALTER TABLE 表名
DROP CONSTRAINT 约束名;   -- 标准SQL
或
ALTER TABLE 表名
DROP CHECK 约束名;        -- MySQL特有
```

##### 7. 修改表名

```sql
ALTER TABLE 旧表名
RENAME TO 新表名;
```

或

```sql
RENAME TABLE 旧表名 TO 新表名;
```

##### 8. 修改表引擎

```sql
ALTER TABLE 表名
ENGINE = 新存储引擎;
```

##### 完整示例：修改 student1 表

```sql
-- 1. 添加出生日期列
ALTER TABLE student1
ADD COLUMN Sbirth DATE AFTER Sname;

-- 2. 修改姓名字段长度
ALTER TABLE student1
MODIFY COLUMN Sname VARCHAR(30) NOT NULL;

-- 3. 添加主键约束（如果未设置）
ALTER TABLE student1
ADD PRIMARY KEY (Sid);

-- 4. 添加唯一约束
ALTER TABLE student1
ADD UNIQUE (Sid);

-- 5. 重命名表
ALTER TABLE student1
RENAME TO students;
```

#### 3.3.1 单表查询

##### 1.选择表中的若干列

```sql
-- 查询全体学生的姓名及其年龄
select Sname,(extract(year from curdate())-extract(year from Sbirthdate)) 年龄
from student;
```

##### 2.LIMIT字句

```sql
-- 保留前四条数据
select *
from countrylanguage
order by CountryCode
limit 4;

-- 去掉四条记录，保留五条数据
select *
from countrylanguage
order by CountryCode
limit 4,5;
```
