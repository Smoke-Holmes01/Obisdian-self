### 💡 使用说明
1. **平台适配**：
   - GitHub：需使用独立行公式（`$$...$$`），行内公式需添加配置支持
   - Typora/VSCode（需数学插件）：完美支持行内和独立公式
2. **扩展用法**：
   ```markdown
   $$ 
   \boxed{\int_a^b f(x)dx = F(b) - F(a)} \tag{公式标签}
   $$
   ```
   - `\boxed{}`：加框强调公式
   - `\tag{}`：公式编号（如 `(1.2)`）
3. **实战建议**：
   - 复杂公式用 [Online LaTeX Equation Editor](https://latex.codecogs.com) 实时预览
   - 希腊字母用 `\` 前缀输入（如 `\alpha` → α）
   - 空格使用 `\,` (小) `\:` (中) `\;` (大) 或 `\quad` `\qquad`

> 此表格兼容绝大多数支持 LaTeX 的 Markdown 编辑器（Typora、VSCode+Markdown All in One 等），在 GitHub 需要替换部分语法（如矩阵环境改为 `\begin{aligned}`）
