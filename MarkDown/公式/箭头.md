### 💡 使用技巧参考表

| **场景**            | **推荐符号**                  | **示例代码**                                   | **说明**                              |
|---------------------|-----------------------------|---------------------------------------------|---------------------------------------|
| 逻辑证明            | `\Leftrightarrow`           | `$A \Leftrightarrow B$`                     | 等价关系证明                          |
| 化学平衡            | `\leftrightharpoons`        | `$\ce{N2 + 3H2 <=>T[高温] 2NH3}$`           | 化学反应可逆过程                      |
| 同构映射            | `\longleftrightarrow`       | `$V \longleftrightarrow W$`                 | 线性空间同构                          |
| 循环系统            | `\circlearrowright`         | `$\text{感知} \circlearrowright \text{行动}$`| 反馈循环系统                          |
| 极限等价            | `\xleftrightarrow{\Delta}`  | `$f(x) \xleftrightarrow{\Delta x \to 0} df$`| 无穷小等价关系                        |
| 双向蕴涵            | `\Leftrightarrow` + `\Box`  | `$p \Leftrightarrow \Box q$`                | 模态逻辑双条件                        |

### ⚠️ 平台兼容性

| **平台/编辑器**       | **支持情况**                              | **替代方案**                                |
|----------------------|------------------------------------------|-------------------------------------------|
| **Typora/VSCode**    | ✅ 全支持 (需启用数学模块)                 | -                                         |
| **LaTeX 文档**       | ✅ 完美支持                               | 加载 `\usepackage{amsmath, amssymb}`      |
| **Jupyter Notebook** | ✅ 全支持                                 | -                                         |
| **GitHub Markdown**  | ⚠️ 仅基础符号 (不含 xxxrightarrow)         | 使用 `\Leftrightarrow` 或组合符号         |
| **Obsidian**         | ✅ 全支持 (需安装 Latex Suite 插件)        | -                                         |
| **微信/知乎**        | 🚫 不支持                                  | 用图片替代或纯文本 `<=>`                  |
