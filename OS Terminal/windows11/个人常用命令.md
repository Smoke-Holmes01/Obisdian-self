# 个人常用命令

## Conda

```powershell
# 初始化 conda 到 PowerShell
conda init powershell

# 从 PowerShell 中移除 conda 初始化
conda init --reverse powershell
```

## npm

```powershell
# 使用淘宝镜像源安装指定包
npm install <package-name> --registry=https://registry.npmmirror.com

# 使用淘宝镜像源全局安装 Claude Code
npm install -g @anthropic-ai/claude-code --registry=https://registry.npmmirror.com
```

## PowerShell

```powershell
# 查看 PowerShell 版本信息
$PSVersionTable
```

## Git

```powershell
# 初始化本地仓库
git init

# 暂存所有变更
git add .

# 提交并附上说明
git commit -m "提交信息"

# 推送到远程 master 分支
git push origin master

# 重命名默认分支为 main
git branch -M main

# 添加远程仓库地址
git remote add origin https://github.com/你的用户名/仓库名.git

# 推送到远程并设置上游分支
git push -u origin main
```
