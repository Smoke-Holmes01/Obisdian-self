## vim常用操作

好的，作为 Linux 专家，掌握 Vim 是必备技能。以下是**最常用、最核心**的 Vim 操作用法，涵盖了日常编辑的绝大部分需求：

**核心概念：模式 (Modes)**

1.  **正常模式 (Normal Mode)**：启动 Vim 后的默认模式。用于移动光标、执行命令（复制、粘贴、删除、保存等）。按 `Esc` 键总是可以回到正常模式。
2.  **插入模式 (Insert Mode)**：用于输入和编辑文本。从正常模式进入插入模式的常用方式：
    *   `i`：在光标前插入
    *   `a`：在光标后插入
    *   `o`：在当前行下方插入新行并进入插入模式
    *   `O`：在当前行上方插入新行并进入插入模式
    *   `I`：移动到当前行行首并进入插入模式
    *   `A`：移动到当前行行尾并进入插入模式
3.  **可视模式 (Visual Mode)**：用于选择文本块。
    *   `v`：进入字符可视模式（按字符选择）
    *   `V`：进入行可视模式（按行选择）
    *   `Ctrl+v`：进入块可视模式（矩形选择）
4.  **命令模式 (Command-Line Mode)**：用于执行保存、退出、查找替换等高级命令。在正常模式下按 `:` 进入。

### 一、移动光标

*   **基本移动：**
    *   `h`：左
    *   `j`：下
    *   `k`：上
    *   `l`：右
    *   `w` / `W`：移动到下一个单词开头 (w: 按单词，W: 按字串，忽略标点)
    *   `b` / `B`：移动到上一个单词开头
    *   `e` / `E`：移动到下一个单词结尾
    *   `0` (数字零)：移动到行首
    *   `^`：移动到行首第一个非空白字符
    *   `$`：移动到行尾
    *   `gg`：移动到文件第一行
    *   `G`：移动到文件最后一行
    *   `:[行号]` + `Enter`：跳转到指定行号 (如 `:10` 跳到第 10 行)
    *   `Ctrl+f`：向下翻一页
    *   `Ctrl+b`：向上翻一页
    *   `Ctrl+d`：向下翻半页
    *   `Ctrl+u`：向上翻半页
    *   `H`：移动到当前屏幕顶部 (High)
    *   `M`：移动到当前屏幕中间 (Middle)
    *   `L`：移动到当前屏幕底部 (Low)
*   **查找移动：**
    *   `/pattern`：向下搜索 `pattern` (按 `n` 下一个，`N` 上一个)
    *   `?pattern`：向上搜索 `pattern`
    *   `f[字符]`：在当前行向右移动到下一个 `[字符]` 处
    *   `F[字符]`：在当前行向左移动到上一个 `[字符]` 处
    *   `t[字符]`：在当前行向右移动到下一个 `[字符]` 的前一个字符处
    *   `T[字符]`：在当前行向左移动到上一个 `[字符]` 的后一个字符处
    *   `;`：重复上一次 `f`, `F`, `t`, `T` 命令
    *   `,`：反向重复上一次 `f`, `F`, `t`, `T` 命令
*   **匹配移动：**
    *   `%`：在配对的括号 `()`, `{}`, `[]` 之间跳转

### 二、编辑文本

*   **删除 (Normal Mode)：**
    *   `x`：删除光标下的字符
    *   `X`：删除光标前的字符
    *   `dw`：删除从光标到下一个单词开头
    *   `d$` / `D`：删除从光标到行尾
    *   `d0`：删除从光标到行首
    *   `dd`：删除当前行
    *   `[n]dd`：删除当前行开始的 `n` 行 (如 `3dd` 删除 3 行)
    *   **在可视模式 (Visual Mode) 下**：选中文本后按 `d` 删除。
*   **复制 (Yank) / 粘贴 (Put) (Normal Mode)：**
    *   `yy`：复制 (yank) 当前行
    *   `[n]yy`：复制当前行开始的 `n` 行
    *   `yw`：复制从光标到下一个单词开头
    *   `y$`：复制从光标到行尾
    *   `p`：在光标**后**粘贴 (如果复制的是整行，则粘贴到下一行)
    *   `P`：在光标**前**粘贴 (如果复制的是整行，则粘贴到上一行)
    *   **在可视模式 (Visual Mode) 下**：选中文本后按 `y` 复制，然后 `p`/`P` 粘贴。
*   **撤销 / 重做 (Normal Mode)：**
    *   `u`：撤销上一次操作
    *   `Ctrl+r`：重做 (恢复被撤销的操作)
*   **替换 (Normal Mode)：**
    *   `r[字符]`：替换光标下的单个字符为 `[字符]`
    *   `R`：进入替换模式 (连续替换，类似覆盖输入，按 `Esc` 退出)
    *   `cw`：删除从光标到单词结尾并进入插入模式 (Change Word)
    *   `c$` / `C`：删除从光标到行尾并进入插入模式
    *   `cc` / `S`：删除整行并进入插入模式
    *   `s`：删除光标下的字符并进入插入模式
    *   `~`：切换光标下字符的大小写
*   **缩进 (Normal Mode / Visual Mode)：**
    *   `>>`：当前行向右缩进 (通常是增加一个 Tab 或空格)
    *   `<<`：当前行向左缩进 (减少缩进)
    *   **在可视模式 (Visual Mode) 下**：选中多行后按 `>` 或 `<` 进行批量缩进。

### **三、查找与替换 **

*   `:%s/old/new/g`：在整个文件 (`%`) 范围内 (`s`ubstitute) 查找 `old` 并替换为 `new`，`g` 表示每行所有匹配项 (global)。不加 `g` 则每行只替换第一个匹配项。
*   `:%s/old/new/gc`：同上，但每次替换前询问确认 (`c`onfirm)。
*   `:10,20s/old/new/g`：在第 10 行到第 20 行范围内替换。
*   `:s/old/new/g`：仅在当前行替换。

### **四、文件操作 **

*   `:w`：保存文件 (Write)
*   `:w [文件名]`：另存为指定文件名
*   `:q`：退出 Vim (Quit) (如果文件未修改)
*   `:q!`：强制退出，不保存修改
*   `:wq` 或 `:x` 或 `ZZ` (Normal Mode)：保存并退出
*   `:e [文件名]`：打开另一个文件进行编辑 (Edit)
*   `:r [文件名]`：将指定文件的内容读取并插入到光标下方 (Read)

### **五、窗口与标签页 **

*   **分割窗口：**
    *   `:sp [文件名]`：水平分割窗口 (`sp`lit)，可加文件名在新窗口打开文件。
    *   `:vsp [文件名]` 或 `:vs [文件名]`：垂直分割窗口 (`v`ertical `sp`lit)。
    *   `Ctrl+w s`：水平分割当前窗口。
    *   `Ctrl+w v`：垂直分割当前窗口。
*   **切换窗口：**
    *   `Ctrl+w w`：循环切换到下一个窗口
    *   `Ctrl+w h/j/k/l`：切换到 左/下/上/右 的窗口
*   **关闭窗口：**
    *   `:q` 或 `:close`：关闭当前窗口
    *   `Ctrl+w c`：关闭当前窗口
    *   `Ctrl+w o`：关闭除当前窗口外的所有其他窗口 (`o`nly)
*   **调整窗口大小：**
    *   `Ctrl+w +`：增加当前窗口高度
    *   `Ctrl+w -`：减少当前窗口高度
    *   `Ctrl+w >`：增加当前窗口宽度
    *   `Ctrl+w <`：减少当前窗口宽度
    *   `Ctrl+w =`：使所有窗口等宽等高
*   **标签页 (Tabs)：**
    *   `:tabnew [文件名]`：在新标签页中打开文件 (或新建空白缓冲区)
    *   `:tabc`：关闭当前标签页 (`tabclose`)
    *   `:tabo`：关闭除当前标签页外的所有标签页 (`tabonly`)
    *   `gt` / `gT`：切换到 下一个 / 上一个 标签页 (`g`o to `t`ab)
    *   `[n]gt`：切换到第 `n` 个标签页 (如 `2gt` 切换到第 2 个标签页)

### **六、其他实用技巧**

*   **重复操作 (Normal Mode)：** 在正常模式下，任何命令前加数字 `n` 表示重复该命令 `n` 次。例如：
    *   `5j`：向下移动 5 行
    *   `3dd`：删除 3 行
    *   `10x`：删除 10 个字符
    *   `2p`：粘贴 2 次
*   **点命令 `.` (Normal Mode)：** 重复上一次的**编辑操作**（在插入模式中输入的内容，或 `d`, `c`, `s`, `x` 等命令及其移动组合）。非常强大！
*   **宏录制 (Normal Mode)：**
    1.  `q[寄存器名]`：开始录制宏到指定寄存器 (如 `qa` 录制到寄存器 `a`)
    2.  执行一系列操作
    3.  `q`：停止录制
    4.  `@[寄存器名]`：执行录制的宏 (如 `@a`)
    5.  `[n]@@`：重复执行上一次执行的宏 `n` 次
*   **显示行号：** `:set number` (或 `:set nu`)。关闭 `:set nonumber` (`:set nonu`)。
*   **语法高亮：** `:syntax on`。关闭 `:syntax off`。
*   **配置文件 (`~/.vimrc`)：** 这是 Vim 的用户配置文件，可以设置各种选项、快捷键映射、加载插件等。强烈建议配置。
*   **插件管理器：** 如 `vim-plug`, `Vundle`, `Pathogen` 等，用于管理 Vim 插件，极大扩展功能 (代码补全、语法检查、Git集成、主题等)。

**最重要的提示：**

1.  **时刻知道自己在哪个模式！** 按 `Esc` 是万能的“回城”键。
2.  **多练习！** Vim 的熟练度完全建立在肌肉记忆上。从基础命令开始，逐步掌握更高级的功能。
3.  **善用帮助：** `:help [主题]` 是 Vim 内置的强大帮助系统。例如 `:help w`, `:help :s`, `:help motion.txt`。

这份列表涵盖了 Vim 最核心和常用的操作。熟练掌握这些，你就能高效地在 Linux 终端中进行文本编辑了。熟能生巧！

# sed

当然！**sed**（Stream Editor）是 Linux 下强大的流式文本处理工具，主要用于 **查找替换、编辑文本** 而无需手动打开文件。以下是 **最常用、最实用的 sed 命令**，涵盖日常文本处理需求：

---

## **🔹 核心语法**
```bash
sed 's/查找内容/替换内容/[选项]' 文件名
```
- `s`：替换命令（substitute）
- `查找内容` 和 `替换内容` 支持 **正则表达式**
- `[选项]`：常用的有 `g`（全局替换）、`i`（忽略大小写）、`p`（打印匹配行）

---

## **🔹 常见用法（常用场景）**

### **1. 基本替换（常用）**
#### **(1) 替换每行的首个匹配**
```bash
sed 's/old/new/' file.txt
```
👉 把 `file.txt` 中每一行的 **第一个** `old` 替换为 `new`（不修改原文件）。

#### **(2) 全局替换（所有匹配项）**
```bash
sed 's/old/new/g' file.txt
```
👉 使用 `g`（global）替换 **所有** `old` 为 `new`。

#### **(3) 替换特定行**
```bash
sed '2s/old/new/' file.txt     # 仅第2行
sed '2,5s/old/new/' file.txt   # 第2-5行
sed '2,$s/old/new/' file.txt   # 第2行到最后一行
```

#### **(4) 替换时忽略大小写**
```bash
sed 's/old/new/i' file.txt
```
👉 `i` 选项使匹配 **不区分大小写**（Case Insensitive）。

---

### **2. 删除行**
#### **(1) 删除匹配的行**
```bash
sed '/pattern/d' file.txt
```
👉 删除所有包含 `pattern` 的行（**d**elete）。

#### **(2) 删除第N行**
```bash
sed '3d' file.txt    # 删除第3行
sed '3,5d' file.txt  # 删除3-5行
sed '$d' file.txt    # 删除最后一行
```

---

### **3. 打印特定行**
#### **(1) 打印匹配的行**
```bash
sed -n '/pattern/p' file.txt
```
👉 `-n` 禁止默认输出，`p` 只打印匹配的行（类似 `grep`）。

#### **(2) 打印指定行**
```bash
sed -n '3p' file.txt      # 打印第3行
sed -n '3,5p' file.txt    # 打印3-5行
sed -n '$p' file.txt      # 打印最后一行
```

---

### **4. 插入 & 追加行**
#### **(1) 在某行前插入**
```bash
sed '2i\This is inserted line' file.txt
```
👉 **i**nsert，在第2行 **前** 插入一行 `This is inserted line`。

#### **(2) 在某行后追加**
```bash
sed '2a\This is appended line' file.txt
```
👉 **a**ppend，在第2行 **后** 追加一行 `This is appended line`。

---

### **5. 替代字符分隔符**
`/` 在 sed 中是默认分隔符，但如果文本包含 `/`，可以用其他字符替代：
```bash
sed 's@/path/to/old@/path/to/new@g' file.txt
sed 's#old#new#g' file.txt
```
👉 `@` 或 `#` 等其他符号可避免冲突。

---

### **6. 替换并生效（直接修改文件）**
#### **(1) `-i` 选项（原地修改）**
```bash
sed -i 's/old/new/g' file.txt
```
👉 `<**i**n-place>` 直接修改 `file.txt`（⚠️慎用！建议先测试）。

#### **(2) 备份原文件**
```bash
sed -i.bak 's/old/new/g' file.txt
```
👉 修改 `file.txt`，并备份原文件为 `file.txt.bak`。

---

## **🔹 正则表达式扩展**
sed 支持正则（PCRE），常见用法：
```bash
sed 's/^#//' file.txt       # 去掉行首的 `#`（注释）
sed 's/[0-9]//g' file.txt   # 删除所有数字
sed 's/\s\+/ /g' file.txt   # 多个空格变为1个空格
sed '/^$/d' file.txt        # 删除空行
```

---

## **🔹 sed + 管道（组合其他命令）**
```bash
cat log.txt | sed 's/error/ERROR/g' | grep "ERROR"
ps aux | sed '/grep/d'     # 过滤掉 `grep` 相关的进程
```

## **📌 总结**
| 需求             | sed 命令               |
| ---------------- | ---------------------- |
| **替换**         | `sed 's/old/new/g'`    |
| **删除行**       | `sed '/pattern/d'`     |
| **打印行**       | `sed -n '/pattern/p'`  |
| **插入行**       | `sed '3i\new line'`    |
| **追加行**       | `sed '3a\new line'`    |
| **多行范围**     | `sed '2,5s/old/new/g'` |
| **直接修改文件** | `sed -i 's/old/new/g'` |

🎯 **关键记忆点**：
1. **`s` 用于替换**（最常用）
2. **`g` 全局匹配**（不加 `g` 只替换每行第一个）
3. **`-n` + `p` 筛选行**（类似 `grep`）
4. **`-i` 直接修改文件**（小心使用）
5. **正则表达式扩展**（`^` `$` `\s` `\+` 等）

掌握这几种 sed 用法，80% 的日常文本处理都能搞定！🎉
