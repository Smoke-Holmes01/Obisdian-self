# 个人常用命令

> [!NOTE]
> 此页保存个人开发环境中的高频操作。尖括号内的内容为占位符，执行前请替换为实际值。

## Conda

| 用途 | 命令 | 备注 |
| --- | --- | --- |
| 初始化 Conda 到 PowerShell | conda init powershell | 修改 PowerShell 启动配置 |
| 移除 PowerShell 初始化 | conda init --reverse powershell | 用于撤销上述配置 |

## npm

| 用途 | 命令 | 备注 |
| --- | --- | --- |
| 使用淘宝镜像安装指定包 | npm install &lt;package-name&gt; --registry=https://registry.npmmirror.com | 临时指定镜像源 |
| 使用淘宝镜像全局安装 Claude Code | npm install -g @anthropic-ai/claude-code --registry=https://registry.npmmirror.com | 需要全局安装权限 |

## PowerShell

| 用途 | 命令 | 备注 |
| --- | --- | --- |
| 查看 PowerShell 版本信息 | $PSVersionTable | PowerShell 7 的版本字段为 PSVersion |

## Git

~~~powershell
# 初始化本地仓库
git init

# 暂存所有变更
git add .

# 提交并附上说明
git commit -m "提交信息"

# 重命名默认分支为 main
git branch -M main

# 添加远程仓库地址
git remote add origin https://github.com/你的用户名/仓库名.git

# 推送到远程并设置上游分支
git push -u origin main
~~~

| 用途              | 命令                     | 备注                    |
| --------------- | ---------------------- | --------------------- |
| 推送到远程 master 分支 | git push origin master | 仅用于远程分支确实为 master 的仓库 |

## WSL

~~~powershell
# 导出 Ubuntu 发行版为备份文件
wsl --export Ubuntu wsl-ubuntu-backup.tar

# 导入备份到指定安装目录
wsl --import Ubuntu <安装目录> wsl-ubuntu-backup.tar
~~~

> [!TIP]
> 导入时使用一个不存在的目标目录；若需指定 WSL 2，可在命令末尾追加 --version 2。
