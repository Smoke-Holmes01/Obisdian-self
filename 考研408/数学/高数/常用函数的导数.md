# 常用函数的导数公式

## 一、基本初等函数导数

| 函数 $f(x)$ | 导数 $f'(x)$ | 说明 |
|:---:|:---:|:---|
| $C$（常数） | $0$ | |
| $x^a$ | $a \cdot x^{a-1}$ | 幂函数，$a \in \mathbb{R}$ |
| $e^x$ | $e^x$ | |
| $a^x$ | $a^x \ln a$ | $a>0, a\neq 1$ |
| $\ln x$ | $\dfrac{1}{x}$ | $x>0$ |
| $\ln\vert x\vert$ | $\dfrac{1}{x}$ | $x \neq 0$ |
| $\log_a x$ | $\dfrac{1}{x \ln a}$ | $a>0, a\neq 1, x>0$ |
| $\sin x$ | $\cos x$ | |
| $\cos x$ | $-\sin x$ | |
| $\tan x$ | $\sec^2 x = \dfrac{1}{\cos^2 x}$ | $x \neq \frac{\pi}{2}+k\pi$ |
| $\cot x$ | $-\csc^2 x = -\dfrac{1}{\sin^2 x}$ | $x \neq k\pi$ |
| $\sec x$ | $\sec x \tan x$ | |
| $\csc x$ | $-\csc x \cot x$ | |
| $\arcsin x$ | $\dfrac{1}{\sqrt{1-x^2}}$ | $\vert x\vert<1$ |
| $\arccos x$ | $-\dfrac{1}{\sqrt{1-x^2}}$ | $\vert x\vert<1$ |
| $\arctan x$ | $\dfrac{1}{1+x^2}$ | |
| $\operatorname{arccot} x$ | $-\dfrac{1}{1+x^2}$ | |

## 二、双曲函数导数

| 函数 | 导数 |
|:---:|:---:|
| $\sinh x$ | $\cosh x$ |
| $\cosh x$ | $\sinh x$ |
| $\tanh x$ | $\operatorname{sech}^2 x = 1 - \tanh^2 x$ |

## 三、求导法则

### 1. 四则运算
- $(u \pm v)' = u' \pm v'$
- $(Cu)' = Cu'$（$C$ 为常数）
- $(uv)' = u'v + uv'$
- $\left(\dfrac{u}{v}\right)' = \dfrac{u'v - uv'}{v^2} \quad (v \neq 0)$

### 2. 复合函数求导（链式法则）
$$y = f[g(x)], \quad \frac{dy}{dx} = f'[g(x)] \cdot g'(x)$$

即：**外层求导后乘以内层导数**

### 3. 反函数求导
$$[f^{-1}(x)]' = \frac{1}{f'[f^{-1}(x)]}$$

### 4. 隐函数求导
方程 $F(x,y)=0$ 两边同时对 $x$ 求导，解出 $y'$。

### 5. 参数方程求导
$$\begin{cases} x = x(t) \\ y = y(t) \end{cases} \quad \Rightarrow \quad \frac{dy}{dx} = \frac{y'(t)}{x'(t)}$$

### 6. 对数求导法
对 $y = f(x)^{g(x)}$ 型函数，先取对数再求导：
$$\ln y = g(x) \ln f(x) \;\Rightarrow\; \frac{y'}{y} = \cdots$$

## 四、高阶导数

| 函数 | $n$ 阶导数 |
|:---:|:---|
| $e^x$ | $e^x$ |
| $a^x$ | $a^x (\ln a)^n$ |
| $\sin x$ | $\sin\left(x + \frac{n\pi}{2}\right)$ |
| $\cos x$ | $\cos\left(x + \frac{n\pi}{2}\right)$ |
| $x^m$ | $m(m-1)\cdots(m-n+1)\,x^{m-n}$ |
| $\ln x$ | $(-1)^{n-1}\,(n-1)!\,x^{-n}$ |
| $\dfrac{1}{x}$ | $(-1)^n\,n!\,x^{-(n+1)}$ |
