| --------- | -------------------- | ----------------------- | ------------------------ | ------------------------------------------- | -------- |
| **算术类**   | `std::plus`          | `std::plus<T>`          | 加法运算 (`arg1 + arg2`)     | `std::plus<int>()(3, 5)`                    | `8`      |

---

### **关键说明：**
1. **头文件依赖**  
   所有函数对象均定义在 `<functional>` 头文件中：
   ```cpp
   #include <functional>
   ```

2. **使用场景**  
   - 可直接实例化为函数对象进行运算（如 `std::plus<int>()`）。
   - 常用作STL算法（如 `std::transform`, `std::sort`, `std::count_if`）的参数：
     ```cpp
     std::sort(vec.begin(), vec.end(), std::greater<int>()); // 降序排序
     ```

3. **透明运算符（C++14起）**  
   可省略类型 `T` 以支持异构类型运算：
   ```cpp
   std::less<>()(5, 3.2);  // 自动推导类型：int + double
   ```

4. **自定义扩展**  
   可通过重载 `operator()` 自定义函数对象，与STL原生函数对象一致使用。
